Ezt az oldalt azért hoztam létre, hogy az esti Szoftverfejlesztő OKJ-s osztályomnak tudjak egy helyet biztosítani, ahol az összes tananyag, órai munka és általam elkészített házi feladat elérhető legyen.
Próbálom rendszeres frissíteni a tartalmakat és minél jobbá tenni az oldalt, de majd az idő eldönti, hogy mennyire tudom ezt teljesíteni.

  • HTML5
  • JavaScript
  • Bootstrap v4.5
  • Font Awesome
  • GitHub
  • GitHub Pages
